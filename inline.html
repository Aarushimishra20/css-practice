Definition:
Inline CSS is used to apply a unique style to a single HTML element.
It is written inside the HTML tag using the style attribute.

✅ Advantages of Inline CSS:

Quick and easy to apply on a single element.

Useful for testing or small design changes.

Styles are applied immediately and have the highest priority.

❌ Disadvantages of Inline CSS:

Not suitable for styling multiple elements or pages.

Makes the HTML code less clean and harder to maintain.

Cannot reuse the same style easily.
