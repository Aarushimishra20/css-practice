What is the Box Model?
Every HTML element is treated as a box on a webpage. This box consists of four layers: content, padding, border, and margin, which together determine the element’s size and spacing.

2. Content:

3. Padding:
The space between the content and the border of an element. It increases the inner spacing of the element.
Example: padding: 10px; → 10px space on all sides of the content.

4. Border:
The border wraps around the padding and content, defining the edge of the element.
Example: border: 2px solid black;

5. Margin:
The space outside the element, used to separate it from other elements.
Example: margin: 20px; → 20px space outside the element.

6. Padding/Margin Shortcut Values:

1 value: All sides the same → padding: 10px;

2 values: Top & Bottom / Right & Left → padding: 10px 20px;

3 values: Top / Right & Left / Bottom → padding: 10px 20px 30px;

4 values: Top / Right / Bottom / Left → padding: 10px 20px 30px 40px;

Order follows clockwise: Top → Right → Bottom → Left

   


   
ontent-box vs Border-box (Interview-ready Definition)

Content-box (default):
Width and height apply only to the content. Padding and border add extra size, increasing the total width/height of the element.

Border-box:
Width and height include content + padding + border, keeping the total size fixed even if padding or border is added.
