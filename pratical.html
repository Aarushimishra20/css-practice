Compared to external CSS, internal CSS has higher priority. This means that if we write a style in the external CSS file and later change that style in the internal CSS, then the browser will ignore the external CSS and apply the internal CSS instead. And if we use all three types (external, internal, and inline) together, then the browser gives first priority to inline CSS, second priority to internal CSS, and third priority to external CSS
